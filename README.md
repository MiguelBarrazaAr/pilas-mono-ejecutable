# pilas-mono-ejecutable

  Ejemplo de crear un ejecutable con pilas engine.
Un ejemplo con música de fondo, y un actor (mono) que puede moverse con las flechas y saltar con el enter.

  Para Ejecutar en windows se debe clickear: jugar.exe

  Versión de pilas simplificada:
* eliminados los actores en su gran mayoría, se dejó solo los mas genéricos.
* eliminadas las imagenes y los sonidos, dejando solo los más requeridos.
